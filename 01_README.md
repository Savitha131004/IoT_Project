# Door Buzzer Using Ultrasonic Sensor

## Project Description
This project develops a touchless door buzzer system using an ultrasonic sensor and Arduino Uno. The system detects the presence of a person approaching the door and automatically activates a buzzer to alert occupants. It provides a safe and contactless method for door notification.

## Aim
The main aim of this project is to design and implement a door buzzer system using an ultrasonic sensor that detects nearby objects and triggers an alert using a buzzer.

## Components Required
- Arduino Uno R3
- Ultrasonic Distance Sensor (4-pin)
- Piezo Buzzer
- Jumper Wires
- Power Supply

## Working Principle
The ultrasonic sensor measures the distance between the object and the sensor using sound waves. When a person comes within a specified distance range, the Arduino processes the signal and activates the buzzer to notify the user.

## Result
The system successfully detected objects within the set distance range and activated the buzzer reliably. It demonstrated effective performance as a touchless doorbell system for safety and convenience.

## Application
- Home security systems
- Contactless doorbell systems
- Smart home automation
- Office entry alert systems
